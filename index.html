Missão de IA: A Ascensão das Funções

Sua tarefa é usar o poder das funções para processar dados críticos e completar a missão.

Objetivo: Analisar o Desempenho dos Robôs

No quartel-general da Cyber-Tech, você recebeu a missão de analisar o desempenho de uma equipe de robôs. Cada robô tem uma pontuação baseada em suas habilidades. Você precisa usar uma função para calcular a pontuação final, aplicando um bônus se o robô tiver mais de 5 anos de serviço.

A Lógica da Função

Uma **função** é um bloco de código que executa uma tarefa específica. Ela nos ajuda a organizar e reutilizar o código. No nosso caso, a função `calcularPontuacaoFinal` recebe dois parâmetros e retorna a pontuação final. 

                
function calcularPontuacaoFinal(pontuacao, anos) {
    let pontuacaoFinal = pontuacao;
    if (anos > 5) {
        pontuacaoFinal += 10; // Adiciona bônus
    }
    return pontuacaoFinal;
}

A função `calcularPontuacaoFinal` é um exemplo perfeito de como encapsular a lógica de negócio em um bloco reutilizável de código.
